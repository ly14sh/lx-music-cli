🎵 LX Music CLI

命令行控制 LX Music 桌面端完整手册

📋 快速开始

安装

npm install -g lx-music-cli

# 或使用 npx
npx lx-music-cli <command>
前提条件：确保 LX Music 桌面端已启动，并在设置中开启"开放 API"（默认端口 23330）

▶️ 播放控制

命令 说明 通信方式
lx play 开始播放 HTTP
lx pause 暂停播放 HTTP
lx toggle 播放/暂停切换 HTTP
lx next 下一首 HTTP
lx prev 上一首 HTTP

🔍 搜索

命令 说明 通信方式
lx search <关键词> 在 LX Music 中打开搜索页面 Scheme

示例

lx search "周杰伦"
lx search "晴天"
lx search "摇滚"
注意：此命令仅打开搜索页面，不会自动播放。需要用户在界面中选择歌曲。

📊 状态查询

命令 说明 通信方式
lx status 显示完整播放状态 HTTP
lx now 显示当前播放的歌曲信息 HTTP
lx lyric 显示当前歌词 HTTP
lx watch 实时监控播放状态（SSE 流） SSE

watch 命令输出示例

🔊 音量控制

命令 说明 通信方式
lx volume <0-100> 设置音量百分比 HTTP
lx mute 静音 HTTP
lx unmute 取消静音 HTTP

示例

lx volume 50    # 设置音量为 50%
lx volume 100   # 最大音量
lx mute         # 静音
lx unmute       # 恢复音量

📁 歌单操作

通过 Scheme URL 打开或播放歌单

命令 说明 通信方式
lx open songlist <source> <id> 打开歌单页面 Scheme
lx play songlist <source> <id> 直接播放歌单 Scheme

示例

# 打开酷我音乐歌单
lx open songlist kw 3373919903

# 播放网易云音乐歌单
lx play songlist wy 123456789

# 支持的 source
kw - 酷我音乐
kg - 酷狗音乐
wy - 网易云音乐
tx - QQ音乐
mg - 咪咕音乐

⚙️ 配置

命令 说明
lx config 显示当前配置（host 和 port）
lx config --host <ip> 修改主机地址
lx config --port <port> 修改端口号

示例

lx config                    # 查看配置
lx config --port 8080        # 修改端口
lx config --host 0.0.0.0     # 允许远程访问
lx config --host 127.0.0.1 --port 23330  # 同时修改

🔗 Scheme URL 详解

LX Music 支持通过 lxmusic:// 协议进行深度链接操作

音乐操作 (music)

# 搜索并播放（需缓存）
lxmusic://music/searchPlay?data={
  "name": "青花瓷",
  "singer": "周杰伦"
}

# 打开搜索页面
lxmusic://music/search?data={
  "keywords": "周杰伦"
}

歌单操作 (songlist)

# 打开歌单
lxmusic://songlist/open?data={
  "source": "kw",
  "id": "3373919903"
}

# 播放歌单
lxmusic://songlist/play?data={
  "source": "kw",
  "id": "3373919903",
  "index": 0
}
支持的 source 值：kw (酷我), kg (酷狗), wy (网易云), tx (QQ), mg (咪咕)

⚠️ 重要说明

关于 searchPlay：
searchPlay 不是真正的搜索，它只能播放已有缓存的歌曲。需要歌曲名、歌手等信息与缓存完全匹配。 建议先用 lx search 在界面中搜索，之后再使用 searchPlay。
通信方式说明：
• HTTP：通过 OpenAPI 端口（默认 23330）发送请求
• Scheme：通过 lxmusic:// 协议调用桌面端
• SSE：Server-Sent Events 实时流，用于 watch 命令
